This is a simple Android application developed as part of the `CTAPDEVL` project. 
The app demonstrates how to set a full-screen background image using Jetpack Compose, ensuring the image scales correctly to fill the device's width.

## ‚ú® Features

- Displays a static background image from the drawable resources.
- Uses **Jetpack Compose** for the entire user interface.
- The background image is scaled to **fill the entire width** of the screen (`ContentScale.FillWidth`) while maintaining its aspect ratio.
- Built with the latest Android development standards.
